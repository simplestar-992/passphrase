## Passphrase

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/passphrase.git
cd passphrase
go build -o passphrase .
```

### Usage

```bash
./passphrase --help
```

---

## Quick Start

```bash
./passphrase [command]
```
