## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/passphrase.git
cd passphrase
go build -o passphrase .
./passphrase --help
```

## 🚀 Quick Start

```bash
./passphrase -h
```

## 🧪 Examples

```bash
# Example 1
./passphrase status

# Example 2
```
