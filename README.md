# PASSPHRASE

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/passphrase.git
cd passphrase
go build -o passphrase .
```

## Usage

```bash
./passphrase --help
```
